# EZLogger WebGL Demo

这是EZLogger日志框架的WebGL演示项目。

## 🌐 在线演示
访问地址: https://github.com/summer198971/ezlogger-webgl-demo

## ✨ 功能特性
- 📋 WebGL平台日志文件管理
- 💾 IndexedDB持久化存储
- 📦 ZIP打包下载
- 🌐 在线文件列表查看
- 🔄 实时日志同步

## 🎮 如何使用
1. 打开在线演示链接
2. 点击"生成测试日志"按钮
3. 使用"显示日志文件列表"查看日志
4. 使用"下载日志ZIP"下载所有日志文件

## 🛠️ 技术栈
- Unity 2022.3 LTS
- EZLogger框架
- WebGL + IndexedDB
- JavaScript桥接
